## Installation
<div align="center">
  <table>
    <tr>
      <th>Installation</th>
      <th>Steps</th>
    </tr>
    <tr>
      <td><b>1. Download</b></td>
      <td>Get the latest JAR from <a href="https://github.com/MCDxAI/reaper-addon/releases">Releases</a> or build from source</td>
    </tr>
    <tr>
      <td><b>2. Install</b></td>
      <td>Place the JAR file in your <code>.minecraft/mods/</code> folder</td>
    </tr>
    <tr>
      <td><b>3. Launch</b></td>
      <td>Start Minecraft with Fabric and Meteor Client installed</td>
    </tr>
    <tr>
      <td><b>4. Configure</b></td>
      <td>Access Reaper settings via the custom Reaper tab in Meteor's GUI</td>
    </tr>
  </table>
</div>
