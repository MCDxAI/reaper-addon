<div align="center">
| Installation | Steps |
| --- | --- |
| **1. Download** | Get the latest JAR from [Releases](https://github.com/MCDxAI/reaper-addon/releases) or build from source |
| **2. Install** | Place the JAR file in your `.minecraft/mods/` folder |
| **3. Launch** | Start Minecraft with Fabric and Meteor Client installed |
| **4. Configure** | Access Reaper settings via the custom Reaper tab in Meteor's GUI |
